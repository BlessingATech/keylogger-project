#  🔒 Python Keylogger (Educational)

A basic keylogger built in Python to understand how keystroke logging works.
Built for cybersecurity learning purposes only; only run on your own machine.

## What it does
- Captures keypresses and logs them to a file with timestamps
- Detects all types of special keys like Shift, Enter, Backspace
- Stops cleanly and abruptedly when ESC is pressed

## How to run it
1. Install the required python library: pip install pynput
2. Run the script: python keylogger.py
3. Press ESC to stop

## What I learned
- How keyloggers work under the hood
- Python event listeners with pynput
- Logging keypresses to a file
- An Ethical use of security tools
